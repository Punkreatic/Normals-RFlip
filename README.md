# Fix Normals Blender Addon

## Description
"Fix Normals" is a Blender addon that automatically fixes normals for all `MESH` objects in the scene. It applies transformations, clears old normals, and makes them consistent and facing outward.

## Installation
1. Download the `fix_normals.py` file.
2. In Blender, go to "Edit > Preferences > Add-ons".
3. Click "Install..." and select the file.
4. Enable the addon from the list of installed addons.

## Usage
1. Open the "N" panel in the "3D View".
2. Go to the "Normals RFlip" tab.
3. Click the "Fix model normals" button to fix the normals for all objects.

## Compatibility
Compatible with Blender 4.0.0 and above.
